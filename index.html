Jukebox Project

This project demonstrates how the project Jukebox works using a serial communication via USART port with an STM32F446RE connected to a computer. It also uses a buzzer to play the notes and the user button to interact with the Jukebox.

This project is intended to be used with the MatrixMCU toolkit. The toolkit is available at GitHub: MatrixMCU.

Version 1

In Version 1, the system works with the user button only. The user button is connected to the pin PC13. The code uses the EXTI13 interrupt to detect the button press.

Parameter Value
Pin PC13
Mode Input
Pull up/ down No push no pull
EXTI EXTI13
ISR EXTI15_10_IRQHandler
Priority 1
Subpriority 0
Debounce time 100-200 ms

You can start from the FSM of Version 1.

Version 2

In Version 2, the system works adds the serial communication. The code uses the USART3 port to receive the commands from the computer. The commands are implemented in Version 4.

The code uses the USART3 port. The port is configured to use the following settings:

Parameter Value
Baudrate 9600
Data bits 8
Stop bits 1
Parity None
Flow control None
USART USART3
Pins PB10 (TX) and PC11 (RX)
Mode Alternative (both pins)
Pull up/ down Pull up (both)
ISR USART3_IRQn
Priority 2
Subpriority 0

You can start from the FSM of Version 2.

Version 3

In Version 3, the system adds the buzzer. The system uses the following pin with PWM configuration to control the buzzer:

Parameter Value
Pin PA6
Timer TIM3
Channel CH1
Frequency (depends on the note playing)
Duty Cycle 50%

To control de duration of the notes, the system uses the TIM2 timer with the following configuration:

Parameter Value
Timer TIM2
Prescaler (depends on the note)
Period (depends on the note)

The system uses the TIM2 interrupt to detect the end of the note. The interrupt is configured with the following settings:

Parameter Value
Interrupt TIM2_IRQHandler
Priority 1
Subpriority 0

You can start from the FSM of Version 3.

Version 4

In Version 4 the system completes its FSM to interact with the user button and the serial communication. In this version it is also managed the sleep mode of the system. The systems sleeps constantly and awakes on the user button press or on the serial communication.

The commands to interact with the Jukebox are:

Command Parameter Description
play - Play the current melody of the Jukebox
pause - Pause the current melody of the Jukebox
stop - Stop the current melody of the Jukebox
next - Play the next melody of the Jukebox
info - Show the information of the current melody of the Jukebox
select melody number Select the melody of the Jukebox. Starting in 0 and as much as MELODIES_MEMORY_SIZE
speed speed Change the speed of the melody. The speed must be greater than 0

Version 5

In Version 5 the user adds its own functions to the Jukebox, such as commands, melodies, new functionalities, new hardware, etc.

Getting Started

Download this project and place it in the projects folder of the MatrixMCU toolkit.

Prerequisites

Be sure that you fulfill the prerequisites of the MatrixMCU toolkit and it is up to date.

If you want to use the serial monitor shown in the video you can to install the VSCode extension Serial Monitor of Microsoft.
