◆ port_usart_init()

void port_usart_init ( uint32_t  usart_id)

Configure the HW specifications of a given USART.

Note
To be successful coding this function, the following steps must be done:

💡 Read the chapter USART: Universal Synchronous and Asynchronous Receiver-Transmitter of the Libro de Fundamentos.
💡 Watch carefully the example project provided in GitHub and the video demo in Youtube.

TO-DO alumnos:

✅ 1. Call function port_system_gpio_config() with the right arguments to configure the USART TX and RX pins as alternate function and no pull up neither pull down connection
✅ 2. Call function port_system_gpio_alternate_function() with the right arguments to configure the USART TX and RX pin as alternate function
✅ 3. Enable the clock for the USART peripheral
✅ 4. Disable the USART ✅ 5. and set the right registers to stablish the configuration: 9600-8-N-1
    💡 Set oversampling to 16
✅ 6. Enable transmission and reception
✅ 7. Disable reception and transmission interrupts
✅ 8. (Given code) Call function NVIC_SetPriority() with the right arguments to set the USART interrupt priority to 2 and the subpriority to 0
✅ 9. (Given code) Call function NVIC_EnableIRQ() with the right arguments to enable the USART interrupt
✅ 10. Enable the USART
✅ 11. Call function _reset_buffer() with the right arguments to reset the input_buffer field of the usart_arr[] element with size USART_INPUT_BUFFER_LENGTH
✅ 12. Call function _reset_buffer() with the right arguments to reset the output_buffer field of the usart_arr[] element with size USART_OUTPUT_BUFFER_LENGTH

Parameters
usart_idUSART ID. This index is used to select the element of the usart_arr[] array
